========================================================================
           TK-Ops-Assistant (TikTok 蓝海运营助手) - 绿色启动版
========================================================================

【简介】
这是一个完全独立的便携版本，无需安装 Python、FFmpeg 或任何其他依赖即可运行。
核心功能：蓝海选品、AI 视频生成、素材下载、收益分析。

【如何运行】
1. 直接双击运行 `tk-ops-assistant.exe`。
2. 程序启动可能需要 3-10 秒进行自解压初始化，请耐心等待。

【目录说明】
- tk-ops-assistant.exe : 主程序
- Output/              : (自动生成) 所有的视频输出、素材下载、Excel 导出都在这里
- Logs/                : (自动生成) 运行日志，遇到问题请打包此文件夹反馈
- .env                 : (可选) 配置文件，如果需要修改 API Key 或其他设置，可在此文件中编辑

【常见问题】
Q: 启动报错 "Failed to execute script main"？
A: 请检查 Logs/startup_limit.log（如果有）。通常是因为杀毒软件误删了临时文件，请尝试将程序目录加入白名单。

Q: 视频生成卡住？
A: 视频生成依赖 CPU/GPU 编解码，请确保电脑没有运行其他高负载程序。

Q: 如何更新？
A: 直接下载新版 EXE 覆盖旧版即可，数据（Output/Logs）不会丢失。
